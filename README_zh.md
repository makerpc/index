
如何使用
1.安装repo 2.使用repo下载。 3.运行打包程序打包。（使用本地文件测试的包） 4.上传打包后的bin文件。 5.在系统上运行，下载到板上。 6.打版本号，进行提交。

所有tinkerforge(创客智造)的索引，目的是中文化并且使用英文补足上游的不足。主要参与人员都是懂英汉双语的

generators  			文档生成器和项目框架生成器
doc				文档，但不包含网站

voltage-current-bricklet	电压电流		四个端子
